Microsoft 보안 공지 MS15-075 - 중요
===================================

OLE의 취약성으로 인한 권한 상승 문제(3072633)
---------------------------------------------

게시된 날짜: 2015년 7월 15일

**버전:** 1.0

요약
----

이 보안 업데이트는 Microsoft Windows의 취약성을 해결합니다. 이 취약성은 임의의 코드가 실행될 수 있는 또 다른 취약성과 결합하여 사용하는 경우 권한 상승을 허용할 수 있습니다. 다른 취약성이 악용되면, 공격자는 이 공지에서 해결된 취약성을 악용해 임의의 코드가 중간 무결성 수준에서 실행되도록 할 수 있습니다.

이 보안 업데이트의 심각도는 지원되는 모든 Microsoft Windows 릴리스에 대해 중요입니다. 자세한 내용은 **영향받는 소프트웨어** 절을 참조하십시오.

이 보안 업데이트는 사용자 입력의 유효성을 검사하는 방식을 수정하여 취약성을 해결합니다. 이 취약성에 대한 자세한 내용은 **취약성 정보** 절을 참조하십시오.

이 업데이트에 대한 자세한 내용은 [Microsoft 기술 자료 문서 3072633](https://support.microsoft.com/ko-kr/kb/3072633)을 참조하십시오.

영향받는 소프트웨어
-------------------

영향받는 소프트웨어 버전 또는 에디션은 다음과 같습니다. 나열되지 않은 버전 또는 에디션은 지원 주기가 끝났거나 영향을 받지 않습니다. 사용 중인 소프트웨어 버전 또는 에디션에 대한 지원 주기를 확인하려면 [Microsoft 지원 주기](http://go.microsoft.com/fwlink/?linkid=21742)를 참조하십시오.

다양한 OLE 권한 상승 취약성
---------------------------

Microsoft Windows OLE가 제대로 사용자 입력의 유효성을 검사하지 못하는 경우 Microsoft Windows OLE에 권한 상승 취약성이 존재합니다. 이 취약성 자체만으로는 임의의 코드 실행이 허용되지 않습니다. 이 취약성은 원격 코드 실행을 허용하는 또 다른 취약성과 결합하여 사용해야 합니다. 이 취약성 악용에 성공한 공격자는 영향받는 시스템에서 제한된 권한을 가진 사용자가 중간 무결성 수준에서 코드를 실행하게 허용할 수 있습니다.

공격자는 또 다른 취약성을 악용하여 임의의 코드를 실행할 수 있습니다. 그럴 경우 공격자는 이 취약성을 악용해 임의의 코드가 중간 무결성 수준(현재 사용자의 권한)에서 실행되도록 할 수 있습니다.

이 보안 업데이트는 사용자 입력의 유효성을 검사하는 방식을 수정하여 취약성을 해결합니다.
