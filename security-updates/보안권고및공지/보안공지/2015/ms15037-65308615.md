Microsoft 보안 공지 MS15-037 - 중요
===================================

Windows 작업 스케줄러의 취약성으로 인한 권한 상승 문제(3046269)
---------------------------------------------------------------

게시된 날짜: 2015년 4월 15일

**버전:** 1.0

요약
----

<span id="sectionToggle0"></span>
이 보안 업데이트는 Microsoft Windows에서 발견된 취약성을 해결합니다. 이 취약성 악용에 성공한 공격자는 알려진 잘못된 작업을 활용하여 작업 스케줄러가 특수 제작된 응용 프로그램을 시스템 계정의 컨텍스트에서 실행되도록 할 수 있습니다. 이렇게 되면 공격자가 프로그램을 설치할 수 있을 뿐 아니라 데이터를 보거나 변경하거나 삭제하거나, 모든 사용자 권한이 있는 새 계정을 만들 수도 있습니다.

이 보안 업데이트의 심각도는 지원되는 모든 버전의 Windows 7 및 Windows Server 2008 R2에 대해 중요입니다. 자세한 내용은 **영향받는 소프트웨어** 절을 참조하십시오.

이 보안 업데이트는 알려진 잘못된 Windows Defender 작업이 영향받는 시스템에 있지 않거나 제거되었는지 확인하여 취약성을 해결합니다. 이 취약성에 대한 자세한 내용은 **취약성 정보** 절을 참조하십시오.

<span id="KBArticle"></span>
이 업데이트에 대한 자세한 내용은 [Microsoft 기술 자료 문서 3046269](https://support.microsoft.com/ko-kr/kb/3046269)를 참조하십시오.

취약성 정보
-----------

<span id="sectionToggle3"></span>
작업 스케줄러 권한 상승 취약성(CVE-2015-0098)
---------------------------------------------

특정 시스템에 있는 알려진 잘못된 작업으로 인해 [작업 스케줄러](https://technet.microsoft.com/ko-kr/library/security/dn848375.aspx)(영문)에 권한 상승 취약성이 존재합니다. 이 취약성 악용에 성공한 공격자는 작업 스케줄러가 특수 제작된 응용 프로그램을 시스템 계정의 컨텍스트에서 실행되도록 할 수 있습니다. 이렇게 되면 공격자가 프로그램을 설치할 수 있을 뿐 아니라 데이터를 보거나 변경하거나 삭제하거나, 모든 사용자 권한이 있는 새 계정을 만들 수도 있습니다.

이 취약성을 악용하려면 공격자가 먼저 대상 시스템에 로그온하고 알려진 잘못된 작업이 시스템에 있는지 여부를 확인해야 합니다. 이 작업이 있는 경우 공격자는 해당 작업을 엔지니어링하여 특수 제작된 응용 프로그램을 시스템 계정의 컨텍스트에서 실행할 수 있습니다. 이 업데이트는 알려진 잘못된 작업이 영향받는 시스템에 있지 않거나 제거되었는지 확인하여 취약성을 해결합니다.

Microsoft는 조정된 취약성 공개를 통해 이 취약성과 관련된 정보를 입수했습니다. Microsoft는 이 보안 공지를 처음에 게시했을 때 이 취약성이 고객을 공격하는 데 공개적으로 사용되었다는 정보를 확인하지 못했습니다.
