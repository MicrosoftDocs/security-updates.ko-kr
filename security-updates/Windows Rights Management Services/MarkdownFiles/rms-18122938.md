RMS 서버 서비스
===============

서버 서비스는 RMS 루트 클러스터에서만 실행됩니다. 서버 서비스는 서버 사용 허가자 인증서 검색을 위해 클라이언트가 온라인 게시를 통해 수행한 요청을 노출합니다.

서버 서비스 응용 프로그램 파일인 Server.asmx는 IIS의 Certification 가상 디렉터리에 있습니다. 관련 항목
