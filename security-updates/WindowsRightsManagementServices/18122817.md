RMS 보안 그룹
=============

RMS 설정은 RMS 서비스 그룹과 고급 사용자 그룹을 만듭니다.

RMS 서비스 그룹은 RMS 작업에 필요한 모든 리소스에 대한 충분한 액세스 권한을 부여받은 로컬 보안 그룹입니다. 관리자는 설치 중에 RMS 서비스 계정으로 사용할 사용자 계정을 지정합니다. 이 사용자 계정은 자동으로 RMS 서비스 그룹 구성원이 되므로 이에 해당하는 사용 권한을 부여 받습니다. RMS는 대부분의 일반 작업 중에 이 사용자 계정으로 실행됩니다.

다른 중요한 RMS 그룹은 고급 사용자 그룹입니다. 이 그룹은 모든 콘텐츠에 대한 모든 권한을 가집니다. 즉, 이 그룹의 구성원은 RMS로 보호된 모든 콘텐츠 파일을 해독하고 모든 RMS 보호를 제거할 수 있습니다. 고급 사용자 그룹에는 기본적으로 구성원이 없으며 RMS 관리자가 자동으로 포함되지 않습니다. 이 그룹의 구성원을 관리하는 일은 RMS로 보호된 콘텐츠의 보안에 중요합니다. 자세한 내용은 이 설명서 모음의 "RMS 서버 운영"에 나오는 "고급 사용자 그룹 사용"을 참조하십시오.
